HTML é uma linguagem de marcação utilizada na construção de páginas na Web, basicamente o esqueleto da página.

O CSS(Cascading Style Sheets) é um mecânismo que adiciona um estilo a um documento web, alterando a cor do texto, fonte e espaçamento entre blocos.

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.
